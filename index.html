How this works
01 uses a conversion of 5.00 points per bust unit.
Cricket uses a conversion of 0.20 MPR per bust unit.

Bust used = (entered score - base average) / conversion.
Max leg score = base average + remaining bust × conversion.

Negative bust is treated as 0, so scoring below your base does not give you extra budget.
